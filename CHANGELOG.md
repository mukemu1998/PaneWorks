# 更新记录

本文档从当前稳定版本开始记录 PaneWorks 的对外发布变化。

## v0.2.4

这一版重点修复工作区恢复和分区保存体验，同时完成一轮源码结构整理，为后续继续扩展窗口吸附、工作区方案和运行时联动能力打基础。

### 优化

- 工作区恢复时，Explorer 文件夹绑定会优先按文件夹路径匹配，减少误吸附其他资源管理器窗口的情况
- 工作区绑定图标改为显示在对应区域左上角，减少遮挡区域中心内容
- 新建分区首次保存时改为弹出命名窗口，避免直接保存为默认名称
- 分区命名弹窗、确认提示弹窗统一为 PaneWorks 圆角暗色风格
- 关于窗口的版本更新要点改为面向普通用户的功能变化说明

### 修复

- 修复两个不同文件夹同时存在时，工作区恢复可能吸附到错误 Explorer 窗口的问题
- 修复新建分区后点击“保存分区”时命名流程不够明确的问题
- 修复部分确认弹窗视觉风格与主菜单不一致的问题

### 维护

- 新增中文开发规范文档，明确后续开发的分层边界、文件职责、行数红线、版本策略和本地测试优先流程
- 拆分主窗口、画布、托盘、工作区恢复、窗口绑定、运行时联动和底层窗口服务中的高复杂度代码，降低后续过度耦合的风险

## v0.2.3

这一版是 v0.2.2 之后的补丁打磨版本，重点整理关于窗口、设置窗口、主菜单标题、侧边悬浮条和源码维护结构。本版不改变核心吸附布局逻辑，主要提升可用性、展示完整度和后续维护性。

### 新增

- 关于窗口新增“当前版本更新要点”区域，可直接查看本版本核心变化
- 关于窗口新增检查更新入口，可读取 GitHub 最新 Release 并引导到发布页
- 主菜单外壳逻辑拆分到 `MainWindow.Workbench.cs`，降低主窗口主文件继续膨胀的维护风险

### 优化

- 主菜单标题 `PaneWorks` 与项目图标配色保持一致，并移除中间空格
- 设置与关于窗口统一为深色圆角风格，窗口尺寸更紧凑
- 设置、关于、主菜单顶部按钮图标改为更稳定的矢量居中显示
- 侧边悬浮条支持拖动，并按当前屏幕边界处理左右与上下贴边
- README 增加源码维护说明，明确源码分层和本地构建产物边界

### 修复

- 修复侧边悬浮条拖到上下屏幕边界后可能被吸到虚拟桌面外的问题
- 修复关于窗口“检查更新”图标使用文字符号导致视觉不协调的问题
- 修复设置/关于窗口部分顶部区域无法拖动的问题

## v0.2.2

这一版重点补强已吸附窗口在工作过程中的临时拓扑调整：窗口退出吸附后，剩余窗口可以通过拖动边界重新靠近相邻窗口或屏幕平行线，自动恢复联动关系，同时避免拖动越过已对齐的参考线。

### 新增

- 运行时边缘拖动支持临时拓扑吸附，剩余窗口边界靠近可连接窗口时会自动重新建立联动
- 支持拖动边界靠近屏幕边缘或同向平行参考线时吸附对齐，用于补齐退出窗口后的新矩形区域
- 平行参考线吸附后会作为硬边界锁止，边界不能继续被拖过参考线

### 优化

- 优化吸附目标识别，窗口拖到区域边缘附近时也能更稳定识别目标分区
- 手动吸附窗口后会补充置顶一次，降低窗口落位后被其他窗口遮住的概率
- 编辑画布的分割线绘制更干净，减少重合分割线造成的视觉干扰

### 修复

- 修复快速释放窗口时偶尔丢失最后一个吸附目标的问题
- 修复拖动已吸附窗口边界靠近平行拓扑线后还能继续越过的问题
- 整理运行时吸附计时器与画布绘制代码中的格式残留，减少后续维护噪音

## v0.2.1

这一版主要整理工作区方案后的稳定性、窗口绑定体验和主菜单窗口行为，让当前功能线更适合作为手动上传与便携分发版本。

### 新增

- 同一区域支持绑定多个已吸附窗口，并按绑定时的前后层级恢复窗口置顶顺序
- 工作区一键绑定时可记录 Explorer 文件夹路径，适合固定项目文件夹工作流
- 主菜单新增“侧边悬浮条”收起入口，与任务栏最小化、托盘驻留分离

### 优化

- `－` 按钮改为标准最小化到任务栏，`×` 与 `Esc` 统一为退出到托盘
- 设置页和状态提示将 `Esc` 语义改为“收进托盘快捷键”，减少与任务栏最小化混淆
- 工作区列表中空绑定与可用绑定使用不同颜色提示，清除绑定后列表会刷新
- 工作区恢复时会持续补偿大型软件启动后的窗口层级和句柄变化
- 源码首页、版本号和便携包命名统一到 `v0.2.1`

### 修复

- 修复 Notepad 等 WindowsApps 路径启动时可能提示缺少 `Microsoft.UI.Windowing.Core.dll` 的问题
- 修复工作区清除绑定后列表状态不立即更新的问题
- 修复同一区域多窗口恢复时最慢启动完成的窗口可能错误占据顶层的问题
- 修复主菜单最小化、关闭、Esc、侧边收起语义混用的问题

## v0.2.0

这一版进入“工作区方案”阶段，在原有自定义吸附布局基础上，增加窗口绑定、工作区恢复、临时调整区吸附和主菜单流程整理。

### 新增

- 工作区方案与普通吸附布局分离，支持单独保存与切换
- 可从当前吸附状态创建工作区方案，并一键绑定已完全吸附的窗口
- 支持手动绑定窗口到指定区域，绑定列表显示应用图标便于识别
- 已打开的工作区窗口可重新吸附，未打开的软件可启动后自动吸附
- Explorer 文件夹窗口支持按文件夹路径绑定到指定区域
- 新增临时调整区吸附快捷键，运行时边缘联动结果可用于当前会话吸附

### 优化

- 主菜单不再默认进入编辑模式，分区编辑、工作区绑定和普通选择模式分离
- 工作区方案默认折叠为高级编辑，降低日常分区编辑干扰
- 主菜单标题区、按钮区、下拉箭头、快捷键提示和中文文案统一打磨
- 退出分区编辑或工作区绑定后自动清除列表选中状态，减少误解
- 托盘菜单、右键菜单和工作区切换文案统一为中文表达
- 修正 EXE、任务栏和资源管理器使用的 ICO 图标资源，使其颜色与设计 PNG 保持一致
- 便携版主程序文件名改为 `PaneWorks.exe`，减少用户面对的内部项目命名

### 修复

- 优化吸附后拖出窗口的还原手感，减少闪动和掉帧
- 修复工作区自动绑定、清除绑定和托盘操作中可能出现的卡住问题
- 修复多屏幕分辨率不一致时吸附位置偏差的问题
- 修复大型软件启动后窗口句柄切换导致自动吸附丢失的问题
- 清理旧的批量窗口排列入口、空预览重置钩子和临时构建产物

### 已知事项

- 个别特殊窗口的吸附兼容性仍需要继续补强
- 部分交互细节和 UI 文案仍可继续打磨
