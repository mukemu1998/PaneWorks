# 更新记录

本文档记录 PaneWorks 对外发布版本的主要变化。

## v0.1.0

首个公开版本。

### 新增

- 透明全屏桌面布局编辑界面
- 递归分割、删除分割、拖动分割线调整比例
- 布局保存、加载、删除
- 自定义吸附触发键与最小化快捷键
- 托盘最小化与托盘快速切换吸附布局
- 窗口吸附、布局预览、布局切换
- 免依赖 Windows 自包含便携版打包脚本

### 优化

- 主界面集中式编辑面板
- 右键菜单中文化与布局切换能力
- 吸附位置、边框、缝隙问题持续打磨
- 分割线高亮、颜色和拖动吸附反馈增强

### 已知事项

- 多显示器支持仍需继续完善
- 不同类型窗口的吸附兼容性还需要继续补强
- 部分 UI 和文案仍在持续打磨中
