# 更新记录

本文档记录 PaneWorks 对外发布版本的主要变化。

## v0.1.1

这一版是在首个公开版本基础上的一次完整打磨更新，重点放在多屏布局、运行时边缘联动、发布形态和项目对外呈现整理。

### 新增

- 多显示器布局可合并保存到同一个布局文件，并在切换后同时作用于多个屏幕
- 已吸附窗口支持共享边缘运行时联动调整
- 项目图标、窗口图标与托盘图标资源
- 便携版 `.sha256` 校验文件输出

### 优化

- 默认空白布局统一命名为“起始空白布局”
- GitHub 首页 README 调整为更正式的项目展示页
- 便携包根目录统一整理为 `PaneWorks.App.exe`、`README.md`、`VERSION`
- 便携版说明与 GitHub 发布指南同步整理
- 用户自建布局继续保存在 `%AppData%\PaneWorks\`，不进入仓库源码目录

### 修复

- 运行时联动拖动时的最小尺寸锁边与边缘计算稳定性
- 项目图标与首页展示资源缺失导致的对外观感不统一问题

### 已知事项

- 个别特殊窗口的吸附兼容性仍需要继续补强
- 部分交互细节和 UI 文案仍可继续打磨

## v0.1.0

首个公开版本。

### 新增

- 透明全屏桌面布局编辑界面
- 递归分割、删除分割、拖动分割线调整比例
- 布局保存、加载、删除
- 自定义吸附触发键与最小化快捷键
- 托盘最小化与托盘快速切换吸附布局
- 窗口吸附、布局预览、布局切换
- 免依赖 Windows 自包含便携版打包脚本

### 优化

- 主界面集中式编辑面板
- 右键菜单中文化与布局切换能力
- 吸附位置、边框、缝隙问题持续打磨
- 分割线高亮、颜色和拖动吸附反馈增强
- 多显示器布局可合并保存到同一个布局文件，切换布局后可同时作用于多个屏幕
- 吸附窗口拖出布局时立即恢复吸附前尺寸，拖动过程保持稳定流畅
- 已吸附窗口支持共享边缘运行时联动调整，不改写原始吸附布局
- 清理废弃的黄色分割线拖拽层与旧实时改布局 resize 代码，降低后续维护干扰
- GitHub 仓库页面的公开说明与提交流程统一改为中文

### 已知事项

- 不同类型窗口的吸附兼容性还需要继续补强
- 部分 UI 和文案仍在持续打磨中
