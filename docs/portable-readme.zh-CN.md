# PaneWorks 便携版

这是 PaneWorks 的 Windows 自包含便携版。

## 使用方法

1. 解压整个压缩包到任意目录
2. 双击 `PaneWorks.App.exe`
3. 首次运行后，布局和设置会保存到：

```text
%AppData%\PaneWorks
```

## 特点

- 不需要额外安装 .NET Runtime
- 解压即可运行
- 适合直接放到 GitHub Release 分发

## 说明

- 如果被系统拦截，请在文件属性里允许执行
- 推荐在 Windows 10 / 11 下使用
- 如果已有旧版本，直接覆盖程序目录即可；用户布局数据不会放在程序目录里
