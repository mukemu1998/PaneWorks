# PaneWorks v0.1.0 发布说明

PaneWorks 的第一个公开版本，核心目标是把“自定义桌面工作区分割 + 窗口吸附”这条主流程打通。

## 这一版可以做什么

- 在透明全屏编辑层里直接划分桌面区域
- 通过右键菜单快速做二等分、三等分、删除分割
- 拖动分割线调整布局比例
- 保存、加载、删除布局
- 把某套布局设为当前吸附布局
- 拖动窗口时按住自定义快捷键进行吸附
- 在托盘菜单里快速切换吸附布局

## 这版重点完成的内容

- 透明桌面式布局编辑体验
- 吸附布局与编辑布局分离
- 自定义快捷键设置
- 托盘最小化与托盘菜单布局切换
- Windows 自包含免依赖发布包

## 当前仍在继续打磨

- 多显示器支持
- 不同应用窗口的兼容性
- 吸附后边框、阴影、缝隙等细节
- 更成熟的布局绑定与自动恢复能力

## 下载与使用

- 下载便携版 zip
- 解压后直接运行 `PaneWorks.App.exe`
- 不需要额外安装 .NET Runtime
