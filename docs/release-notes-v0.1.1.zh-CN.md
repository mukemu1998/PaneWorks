# PaneWorks v0.1.1 发布说明

PaneWorks `v0.1.1` 是在首个公开版基础上的一次完整打磨更新，重点把多屏布局、运行时边缘联动、项目图标、便携包结构和 GitHub 对外展示统一整理到位。

## 本次更新

### 新增

- 多显示器布局可合并保存到同一个布局文件，并在切换后同时作用于多个屏幕
- 已吸附窗口支持共享边缘运行时联动调整
- 项目图标、窗口图标与托盘图标资源
- 便携版 `.sha256` 校验文件输出

### 优化

- 默认空白布局统一命名为“起始空白布局”
- GitHub 首页 README 调整为更正式的项目展示页
- 便携包根目录统一整理为 `PaneWorks.App.exe`、`README.md`、`VERSION`
- 便携版说明与发布指南同步整理

### 修复

- 运行时联动拖动时的最小尺寸锁边与边缘计算稳定性
- 项目图标与首页展示资源缺失导致的对外观感不统一问题

## 下载与使用

- 下载 `PaneWorks-v0.1.1-win-x64-portable.zip`
- 可选校验 `PaneWorks-v0.1.1-win-x64-portable.zip.sha256`
- 解压后直接运行 `PaneWorks.App.exe`
- 不需要额外安装 .NET Runtime

## 当前仍在继续打磨

- 个别特殊窗口的吸附兼容性仍需要继续补强
- 部分交互细节和 UI 文案仍可继续打磨
